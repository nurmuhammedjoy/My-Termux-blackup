# 🌟 i3wm Dotfiles – Native Termux Setup

A minimal and efficient i3 window manager configuration designed to run natively inside Termux (no PRoot), featuring:

- 🧩 i3wm (via X11 in Termux)
- 🎛️ Polybar for status and system info
- 🔍 Rofi as app launcher and powermenu
- 🎨 Custom GTK theme and icons
- 🧪 Optimized for lightweight, usable X11 desktop on Android

---

## 📦 Features

- ⚡ **Lightweight and Fast**: Runs natively on Android with no proot overhead
- 🎯 **Productive**: Workspaces, keybindings, and power menus streamlined
- 🌈 **Stylish**: GTK theme and icons for a cohesive look
- 📟 **Polybar Modules**: Battery, clock, volume, network, and more
- 🎨 **Rofi**: Themed for consistency with the GTK/UI setup
